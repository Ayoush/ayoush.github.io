2 23-10-2023 242. Valid Anagram

Solution Link 1 So my go to solution for this. The first check if two string length is of different length then there is no chance it would be anagram. After this we will put two hash list and for each element check if the existing element is already there in opposite hash list and then remove it else add it to there hash list. At the end if both the hash list are empty at the end then return true else false String Anagram, Hash
